## Guia al usuario

## *Clonar el repositorio:*

```
 https://github.com/braiandevcode/ConectaServ.git
```


## *Ejecutar comando de instalación*

```
npm install
```

## *Levantar servidor de vite + react en su local:*

```
npm run dev
```
